Chesapeake Bay Data Hub

The Chesapeake Bay Data Hub is an extensive data base holding present and historical data relevant to the Chesapeake Bay and its health. The data hub allows users to request data via their API and covers several different Bay health topics. An example of an API call in Python for historical data of sediment can be seen below. Additionally, a .csv file of the sample data is available for viewing. This database will be utilized to view data topics covering water quality, sediment, toxins, living resources, and more.

Chesapeake Monitoring Cooperative

The Chesapeake Bay Monitoring Cooperative is a compilation of water quality data in the Chesapeake Bay region. The data is downloaded via a query tool on the website and allows the user to choose different options such as timeframes or locations of water quality. The image on the right displays a sample of the data collected from this query tool.

Eyes on the Bay

Eyes on the Bay is another website that allows users to query data. The data offered here is location based and spans many different water measurements such as tempertaure and salinity in one location measured every 15 minutes. The data goes back to 2001 for measurements taken every 15 minutes and has measurements that go back as far as 1985. This data will be very useful in noting changes in the water over time and in different locations.
